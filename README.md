# FlowSafe_Sprint2
Repositório da Sprint 2 do projeto FlowSafe. Ecossistema inteligente de semáforos acessíveis com Arduino UNO. Contém protótipo físico, simulações no Tinkercad, arquitetura de algoritmos (Flowgorithm/Portugol), documentação em normas ABNT e planejamento de gestão ágil (5W2H, Cronograma e Daily Scrum).
